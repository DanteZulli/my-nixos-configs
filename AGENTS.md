# AGENTS.md

Single-host NixOS flake (`lachata`, user `dante`, x86_64-linux).
Built with the [Den Framework](https://den.denful.dev) (Dendritic Pattern).

System: `nixpkgs-unstable`, home-manager: `master` (semi-rolling, `system.stateVersion` = `"26.05"`).

## Commands

```bash
just rebuild        # alejandra --quiet . + nix run .#lachata -- switch
just debug          # same with --show-trace --verbose
just update         # nix flake update
just clean          # keep last 10 system + user generations, then nix-collect-garbage --delete-old
just vm             # run a QEMU VM to test the config
just list           # show all recipes
```

`rebuild`/`debug` run `alejandra` first — no separate format step.

Syntax check a single file (no test framework):
```bash
nix-instantiate --eval --strict <file>
```

Regenerate `flake.nix` after input changes:
```bash
nix run .#write-flake
```


## Architecture

- **Module pattern**: `den.aspects.<name>.<class> = { ... }` where `<class>` is `nixos` or `homeManager`.
- **Enabling**: List the aspect in the entity's `includes` (e.g., `den.aspects.boot`). `includes` are dot-access references, not import paths.
- **No feature-gating**: Den's `includes` replaces `options.*.enable` / `lib.mkIf`.
- **Function-call includes** require parens: `(den.batteries.user-shell "bash")`.
- **Host hardware**: added via `nixos.imports = [../hosts/.../hardware-configuration.nix]` in the host aspect, not via `includes`.
- **Batteries** in `defaults.nix`: `define-user`, `primary-user`, `user-shell`, `hostname` are applied globally via `den.default.includes`.

## Conventions

- Indent: 2 spaces.
- Functions: `{ den, pkgs, ... }:` — destructure only what's used.
- Naming: kebab-case files, camelCase attrs/vars, dot-separated NixOS options.
- Strings: double quotes normally, `''...''` for multi-line.
- Multi-element lists: one per line. Single-element: inline.

## Git

- Remote: `git@github.com:DanteZulli/my-nixos-configs.git`
- Branch: `main`, `push.autoSetupRemote = true` configured.
- Commits: imperative mood, lowercase, no trailing period.
